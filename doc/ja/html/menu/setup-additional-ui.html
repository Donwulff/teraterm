Additional settings / "UI" タブ ([Setup] メニュー)

UI Language
言語ファイル(lang\*.lng)の指定により、メニューやメッセージの言語を切り替えることができます。 言語ファイルも参照してください。 日本語メニューを表示するためには Japanese.lng を選択してください。
ダイアログフォント
ダイアログで使用するフォントを指定します。

ダイアログフォントに極端に大きなサイズや横幅の広いフォントを設定すると、 表示される文字が欠けたり、ダイアログが画面からはみ出して[ok]ボタン等が押せなくなることがあります。

選択 ボタン
OSのフォントダイアログが開き、使用するフォントを選択できます。 開いたとき、現在選択中フォントが選択された状態となっています。
フォントダイアログのフォント一覧に表示するフォントは 下のチェックボックスで変更することができます。
デフォルト ボタン
OSが使用しているダイアログフォントを選択します。
プロポーショナルフォントをフォントダイアログの一覧に表示する
フォントダイアログのフォントの一覧にプロポーショナルフォントを表示します。
選択中フォントがプロポーショナルフォントの場合、 このチェックボックスは自動的にオンになり、変更できなくなります。
OSで非表示のフォントもフォントダイアログの一覧に表示する
フォントダイアログのフォントの一覧に非表示設定されたフォントを表示します。
選択中フォントが非表示フォントの場合、 このチェックボックスは自動的にオンになり、変更できなくなります。

フォントダイアログで特定フォントを非表示とする機能は Windows 7以降で利用可能です。
