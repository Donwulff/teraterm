漢字コード変更プラグイン

TTXKanjiMenu プラグインを入れると、Tera Term のメニューに [漢字コード] メニューが追加されます。
メニュー内の項目から送受信文字コード、Ambiguous Character Width を変更することができます。

メニューから[設定]を選ぶと、このプラグインの設定ダイアログが表示されます。

保存される設定

TERATERM.INI の KanjiMemu セクションに次のキーを保存します。 
| キー                        | 内容                       | 値                  | デフォルト値       |
|-----------------------------|----------------------------|---------------------|--------------------|
| UseOneSetting               | 送受文字コードを同一とする | ON 同一             | ON                 |
|                             |                            | OFF 同一ではない    |                    |
|-----------------------------|----------------------------|---------------------|--------------------|
| Language                    | 文字コードメニュー         | Japanese 日本語向け | OSの言語設定による |
|                             |                            | Korean 韓国語向け   |                    |
|                             |                            | Disable 表示しない  |                    |
|-----------------------------|----------------------------|---------------------|--------------------|
| AmbiguousCharacterWidthMenu | メニューを表示             | ON 表示する         | OFF                |
|                             |                            | OFF 表示しない      |                    |
|-----------------------------|----------------------------|---------------------|--------------------|
| SettingMenu                 | メニューを表示             | ON 表示する         | ON                 |
|                             |                            | OFF 表示しない      |                    |

Language=Disable にするとメニューが出てこなくなるため設定変更などができなくなります。
SettingMenu=OFF にすると設定ダイアログを開けなくなります。
設定を保存すると Tera Term を再起動しても元の状態に戻りません。

表示できるようにするにはTERATERM.INI を編集してください。
次のように変更して、Tera Term を再起動してください。 

    [KanjiMemu]
    SettingMenu=ON
    Language=
