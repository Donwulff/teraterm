KanjiMenu plugin

When TTXKanjiMenu plugin is installed, [KanjiCode] menu is added to Tera Term.
Sending/receiving character code and Ambiguous Character Width can be changed using menus.

Setting dialog for this plugin is opend selecting [Setting] from menu.

Saved Settings

List of keys stored in KanjiMemu section of TERATERM.INI. 
| key                         | Description                                       | value                  | default value |
|-----------------------------|---------------------------------------------------|------------------------|---------------|
| UseOneSetting               | Use same character encoding for sending/receiving | ON  same encoding      | ON            |
|                             |                                                   | OFF  not same encoding |               |
|-----------------------------|---------------------------------------------------|------------------------|---------------|
| Language                    | Character Encoding Menu                           | Japanese  for Japanese | OS setting    |
|                             |                                                   | Korean  for Korean     |               |
|                             |                                                   | Disable Not Displayed  |               |
|-----------------------------|---------------------------------------------------|------------------------|---------------|
| AmbiguousCharacterWidthMenu | Show ambiguous character width menu               | ON  Show menu          | OFF           |
|                             |                                                   | OFF Do not show        |               |
|-----------------------------|---------------------------------------------------|------------------------|---------------|
| SettingMenu                 | Show setting menu                                 | ON  Show menu          | ON            |
|                             |                                                   | OFF Do not show        |               |

If Language=Disable, menu will not appear and you will not be able to change settings.
If SettingMenu=OFF, settings dialog cannot be opened.
Once the settings are saved, The menu is not displayed after restarting Tera Term.

Edit TERATERM.INI to make it visible.
Make following changes and restart Tera Term. 

    [KanjiMemu]
    SettingMenu=ON
    Language=
