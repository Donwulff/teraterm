## TTXChangeFontSize

[OSDN](https://osdn.net/users/doda/pf/TTXChangeFontSize/wiki/FrontPage)

cmakeを使用したビルド時はソースを置くと自動でビルドされる

次のコマンドでソースを取得
```
cd TTXChangeFontSize
git clone git://git.pf.osdn.net/gitroot/d/do/doda/TTXChangeFontSize.git
```
